Media Metadata Cleaner

Remove hidden metadata from your photos and videos—on your own computer.

What does this tool do?

Photos and videos often contain metadata: information stored inside the file, such as the date and time it was taken, the device or camera model, and sometimes location (GPS). This app lets you remove that metadata before you share files, so you don’t accidentally share more than you intend.

Everything runs on your computer. Your files are never uploaded to the internet.

Download

Get the latest version from the Releases page. You’ll see several files; pick the one that matches your system:

  • Windows — choose the file that ends in .msi (installer) or .nsis.exe (setup). Use the x64 version for most PCs.
  • macOS (Apple) — choose the .dmg file. If you have an M1/M2/M3 Mac (Apple chip), use aarch64; for Intel Macs, use x64.
  • Linux — choose .deb (Debian/Ubuntu), .rpm (Fedora/Red Hat), or .AppImage (runs on most distributions).

Installation

Windows

  1. Open the downloaded .msi or .exe file.
  2. Follow the installer (Next → Next). You may need to allow the app if Windows asks.
  3. When it’s done, open Media Metadata Cleaner from the Start menu or desktop.

macOS

  1. Open the downloaded .dmg file.
  2. Drag Media Metadata Cleaner into your Applications folder.
  3. Open it from Applications. The first time, macOS may say it’s from an unidentified developer—click Open in System Settings (or Security & Privacy) to allow it.

Linux

  • .deb (Ubuntu, Debian): Double‑click the file to open the software installer, then click Install. Or in a terminal: sudo dpkg -i media-metadata-cleaner-linux_*_amd64.deb
  • .rpm (Fedora, etc.): Open the file with your package manager, or in a terminal: sudo rpm -i media-metadata-cleaner-linux-*-1.x86_64.rpm
  • .AppImage: Make the file executable (right‑click → Properties → Permissions, or chmod +x filename.AppImage), then double‑click to run.

How to use

  1. Set options (optional). You can leave the defaults. Useful choices:
    • Show media — Leave checked to see image previews; uncheck if you’re cleaning many files and the app feels slow.
    • View metadata only (don’t clean) — Check this if you only want to see what metadata is in the file without changing it.
    • Save cleaned file — By default, the app creates a new file with -cleaned in the name (e.g. photo.jpg → photo-cleaned.jpg). You can change this to overwrite the original or use a random name, and choose which folder to save in.
  2. Click “Choose media files…” and select one or more image or video files. You can select multiple at once.
  3. Click “Process.” The app will clean each file and show a list of results.
  4. Check the results. Each file gets a row:
    • Green — Succeeded.
    • Yellow — Completed with warnings (you can expand the row to see details).
    • Red — Something went wrong (expand the row to see the error).
    Click a row to expand it and see the original vs cleaned metadata, or any errors or warnings.
  5. Use the Search box above the results to filter by filename or any text in the messages.

Your cleaned files are saved according to the options you chose (by default, in the same folder as the originals, with -cleaned in the name).

Tips and notes

  • All work is done on your machine. Nothing is sent to the internet.
  • Use the correct file extension for your image (e.g. a JPEG should end in .jpg or .jpeg, not .png).
  • If something doesn’t work or you have a suggestion, you can report it or open an issue on GitHub.

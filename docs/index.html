What does this tool do?

Photos and videos contain metadata: information stored inside the file, such as the date and time it was taken, the device or camera model, and sometimes location (GPS). This app lets you remove that metadata before you share files.

Everything runs on your computer. Your data or files are never uploaded to the internet.

The app is free and open source. You can view the source code on GitHub, and the code can be audited by anyone to make sure it's not doing anything nefarious.

Installation

Windows

  1. Download the installer (msi).
  2. Open the downloaded installer file.
  3. If windows presents the blue window with the text "Windows protected your PC":
    • click "More info"
    • click "Run anyway."
    • (Windows did not protect your PC, I just did not pay Microsoft to disable this warning)
  4. Follow the installer instructions.
  5. Open "Media Metadata Cleaner" from the Start menu or desktop.

macOS

  1. Download the installer (dmg).
  2. Open the downloaded .dmg file.
  3. Drag Media Metadata Cleaner into your Applications folder.
  4. Open it from Applications.
  5. The first time, macOS may say it’s from an unidentified developer:
    • Open Settings
    • click "Privacy and Security"
    • scroll down to Security
    • click "Open Anyway" under "media-metadata-cleaner" was blocked from use.

Linux

  • Ubuntu, Debian:
    • Download the .deb package
    • double‑click it to open the software installer and click Install
    • Or in a terminal: sudo dpkg -i media-metadata-cleaner-linux_*_amd64.deb
  • Fedora, Red Hat, CentOS, etc.:
    • Download the .rpm package
    • open it with your package manager
    • Or in a terminal: sudo rpm -i media-metadata-cleaner-linux-*-1.x86_64.rpm
  • Other Linux distributions:
    • Download the .AppImage file
    • make it executable (right‑click → Properties → Permissions, or chmod +x filename.AppImage)
    • then double‑click to run

How to use

  1. Set options
    • Show media — Leave checked to see image previews; uncheck if you’re cleaning many files and the app feels slow.
    • View metadata only (don’t clean) — Check this if you only want to see what metadata is in the file without changing it.
    • Save cleaned file — Choose the name and destination folder for the cleaned file.
  2. Click “Choose media files…” and select one or more image or video files. You can select multiple at once.
  3. Click “Process.” The app will process each file and show a list of results.
  4. Check the results. Each file gets a row:
    • Green — Succeeded.
    • Yellow — Completed with warnings (you can expand the row to see details).
    • Red — Something went wrong (expand the row to see the error).
    Click a row to expand it and see the original vs cleaned metadata, or any errors or warnings.
  5. Use the Search box above the results to filter by filename or any text in the messages.

Your cleaned files are saved according to the options you chose (by default, in the same folder as the originals, with -cleaned in the name).

Notes

  • Use the correct file extension for your image (e.g. a JPEG should end in .jpg or .jpeg, not .png).
  • If something doesn’t work or you have a suggestion, you can report it or open an issue on GitHub.
